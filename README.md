# Task Manager Pro

A professional command-line Task Manager application built with Python, SQLite, and Object-Oriented Programming.

---

## Features

- ✅ Add Task
- ✅ View All Tasks
- ✅ Search Tasks
- ✅ Update Tasks
- ✅ Delete Tasks
- ✅ Mark Task as Completed
- ✅ Mark Task as Pending
- ✅ SQLite Database
- ✅ Input Validation
- ✅ Clean OOP Architecture

---

## Technologies

- Python 3
- SQLite3
- Object-Oriented Programming (OOP)

## Database

SQLite is used to store all tasks.

Each task contains:

- ID
- Title
- Description
- Status
- Created Date
